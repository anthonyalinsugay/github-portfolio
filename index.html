PROJECT 01

AQUA – Davao

PROJECT 02

GERIM System

Gender Equality Reporting & Inclusion Monitoring

Because every voice deserves to be heard — GERIM is a structured platform that empowers individuals to safely report gender-related concerns while giving institutions the tools to monitor, respond, and build more inclusive environments. Aligned with UN SDG 5 on Gender Equality, this system bridges the gap between those who experience bias and the people with the power to change it.

SDG Aligned Reporting System Inclusion Admin Dashboard
